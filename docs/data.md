# Documentación para archivos _data/*.yml

## projects.yml

* `name`: Nombre de tu proyecto
* `descr`: Descripción de tu proyecto
* `demo`: Enlace a tu proyecto
* `tags`: Lista de tecnologías que usaste en tus proyectos
